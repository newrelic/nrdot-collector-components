# Receivers

This directory contains OpenTelemetry Collector receiver components for the New Relic Dot Collector.

## Available Receivers

- **nopreceiver**: Minimal placeholder receiver component
- **newrelicoraclereceiver**: Oracle database monitoring receiver with comprehensive metrics collection including query performance monitoring, RAC support, PDB support, and advanced features
- **newrelicsqlserverreceiver**: SQL Server database monitoring receiver with query performance monitoring, Azure AD authentication, wait event analysis, and comprehensive database metrics
